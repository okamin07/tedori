# TEDORI（テドリ）要件定義

## 1. コンセプト

副業・独立の「手取り」を**シナリオ比較**で決めるツール。Geminiに聞く単発試算ではなく、申告区分・売上増の**差分**を並べて意思決定する。

- ブランド名: **TEDORI**
- ポジション: **副業・独立の「もしも」比較器**
- ターゲット: 副業をしている/始めたい会社員、個人事業主・フリーランス
- 想定デバイス: モバイル70%+（モバイルファースト）

### Gemini を超える理由

| LLM | TEDORI |
|---|---|
| 毎回プロンプト | プリセット1タップ |
| 回答がブレる | テスト済み計算・再現性 |
| 1シナリオ | 申告比較・売上ステップ表 |
| 一般論 | 試算に紐づくCTA1本 |

## 2. マネタイズ設計

| フェーズ | 施策 |
|---|---|
| 初期 | アフィリエイト（会計ソフト freee/マネーフォワード、開業支援、クレカ、税理士紹介） |
| 集客後 | 機能追加＋一部を課金解放（保存・複数シナリオ比較・年間試算・PDF出力・広告非表示） |

アフィリは「ツールの結果 → 次の一手」に自然に接続する（例: 経費を漏れなく計上 → 会計ソフト）。

## 3. ツール要件（MVP）

2つのモードを持つ。

### モードA: フリーランス/個人事業主
入力: 年間売上・経費・申告区分（青色65/55/10・白色）・社会保険料（自動概算＋手動上書き）・その他控除
出力: 手取り、所得税、復興特別所得税、住民税、個人事業税、社会保険料、実効税率、内訳

### モードB: 副業（会社員）
入力: 本業の額面年収・副業所得（売上−経費）・申告区分
出力: 副業によって「増える税金」と「副業の手取り」、本業と合算した限界税率

## 4. 計算ロジック（2025年分・令和7年／概算）

- 所得税 = 課税所得 × 税率 − 速算控除（7段階 5〜45%）
- 復興特別所得税 = 所得税 × 2.1%
- 住民税 = 課税所得（住民税基礎控除43万）× 10% + 均等割5,000円
- 給与所得控除: 2020年以降の速算表
- 基礎控除: 所得税48万 / 住民税43万（合計所得2400万以下）
- 青色申告特別控除: 65万 / 55万 / 10万（事業所得のみ）
- 個人事業税: (事業所得 − 290万) × 5%（該当時のみ、業種簡易）
- 消費税: 免税事業者前提（売上1000万以下）
- 社会保険料: 国民年金（概算）＋国民健康保険（概算・自治体差あり）→ 手動上書き可

すべて「概算・目安」であることを明示。最終額は個別事情・自治体により変動する旨をディスクレーマー表示。

## 5. デザイン方針

- トーン: モバイルの計算機。白ベース・1アクセント（ブランドブルー）
- レイアウト: 手取り固定ヘッダー → 比較 → 入力 → 売上ステップ表
- ビジュアル: 積み上げバーで内訳。文字だけの帳票UIは使わない
- メディア: トップ下部にコンパクト表示（SEO用）
